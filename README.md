# OSSP1_RL
건국대학교 오픈소스소프트웨어프로젝트1 수업의 텀프로젝트 레포입니다. 강화학습을 기반으로 한 랜덤성있는 마리오게임을 플레이하는 것이 목적입니다.

## Mario Game BaseLine Usage
* Baseline Mario Game is forked by here. [super mario game](https://github.com/meth-meth-method/super-mario/)

## Running
* pip3 install -r requirements.txt
* python main.py

## Controls
* Left, Right : Move Left, Right.
* Space : Jump
* Shift : Boost

## Dependencied
* pygame
* scipy
